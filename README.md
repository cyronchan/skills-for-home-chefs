# Skills for Home Chefs

AI skills that help anyone cook at home with more confidence, flavour, and less waste.

This is not a library of rigid recipes.

It is a collection of practical cooking workflows for real kitchens: small kitchens, imperfect ingredients, busy evenings, limited equipment, tight budgets, family preferences, and leftovers that need using.

## Principle

A good home meal does not require perfection.
It requires good decisions: use what you have, understand flavour, cook safely,
waste less, and make food worth sharing.
