# Skills for Home Chefs

AI skills that help anyone cook at home with more confidence, flavour, and less waste.

This is not a library of rigid recipes. It is a collection of practical cooking workflows for real kitchens: small kitchens, imperfect ingredients, busy evenings, limited equipment, tight budgets, family preferences, and leftovers that need using.

## ➕ Add your recipe

This project grows through contributions. If you cook a dish at home that you love, add it here.

**It takes about 15 minutes and no coding experience is needed.**

1. [Fork this repo](https://github.com/cyronchan/skills-for-home-chefs/fork)
2. Copy [`templates/RECIPE-SKILL-TEMPLATE.md`](templates/RECIPE-SKILL-TEMPLATE.md) into `skills/recipes/[cuisine]/[dish-name]/SKILL.md`
3. Fill it in with your recipe
4. Open a Pull Request

All cuisines, cooking traditions, and budgets are equally welcome.

📝 [Full contribution guide →](CONTRIBUTING.md)

## Principle

A good home meal does not require perfection.
It requires good decisions: use what you have, understand flavour, cook safely, waste less, and make food worth sharing.
